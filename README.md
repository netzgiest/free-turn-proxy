## О проекте

**btp** - туннель UDP/TCP через TURN-реле VK Calls. Клиент берёт временные TURN-учётки из ссылки на звонок и гонит ваш
трафик (WireGuard, Xray/VLESS) до сервера на VPS поверх DTLS.
